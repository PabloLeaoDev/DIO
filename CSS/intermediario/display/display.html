O display: inline, como o nome diz, fica ordenado em linha (horizontalmente), ao contrário do display: block, o qual fica ordenado em coluna (verticalmente). Um aspecto importante dessa propriedade que vale destacar é o fato do width e height não se aplicarem a ele, já o padding e o margin se aplicam, mas somente no eixo X (horizontal).

O display: inline-block combina as características do display: block e do display: inline, já que ele é tão personalizável quanto o block mas mantêm-se ordenado em linha como o inline.

Propriedade display: none

PASSE O MAUSE POR MIM PARA ATIVAR O DISPLAY: NONE

O display: none faz todo o container 'pai' e seus 'filhos' sumirem, mas a tag continua existindo no código-fonte.

A propriedade display: none é similar a outra propriedade chamada visibility: hidden, contudo a primeira tem como função eliminar qualquer rastro de um elemento, já a segunda torna o elemento invisível mas conserva o espaço que ele ocupa/ocupou.
