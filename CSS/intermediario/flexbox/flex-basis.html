A propriedade flex-basis define um valor inicial para um flex-item. Tal propriedade se submete as propriedades min-width/max-width/min-height/max-height. Além disso, ela funciona no eixo principal e possui duas palavras-chave como valor: auto e content.

IMPORTANTE: ela funciona somente nos seletores dos itens, não dos container's
