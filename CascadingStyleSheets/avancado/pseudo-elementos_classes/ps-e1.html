Pseudo-elementos

Usado para adicionar elementos extras na página, sem declará-los no HTML. Também pode ser usado para selecionar uma parte especifíca de um elemento.

Identificado por ::palavra-chave

Pseudo-elemento - ::first-line

Seleciona a primeira linha de um texto.

Propriedades disponívies:

  • color
  • background
  • font
  • Propriedades de texto
