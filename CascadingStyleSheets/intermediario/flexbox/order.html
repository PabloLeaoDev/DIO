A propriedade order define a ordem de um elemento. Por padrão todos os elementos do container-flex estão com order: 0, portanto, o item que estiver com um valor no order < 0, virá primeiro, e quando estiver > 0, virá depois.

Pode-se manipular o valor padrão da propriedade order em um container-flex caso necessário. Além disso, quando algum item estiver com o mesmo valor na propriedade order, eles segirão a sequência do HTML.
