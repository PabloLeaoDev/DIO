1. Tag <h1>

A tag <h1> é usada para definir o título principal de uma página ou seção.

(Voltar)

2. Tag <h2>

A tag <h2> é usada para definir um título de segunda importância.

(Voltar)

3. Tag <h3>

A tag <h3> é usada para definir um título de terceira importância.

(Voltar)

4. Tag <h4>

A tag <h4> é usada para definir um título de quarta importância.

(Voltar)
5. Tag <h5>

A tag <h5> é usada para definir um título de quinta importância.

(Voltar)
6. Tag <h6>

A tag <h6> é usada para definir um título de sexta importância.

(Voltar)

7. Tag <p>

A tag <p> é usada para definir um parágrafo de texto.

(Voltar)

8. Tag <mark>

A tag <mark> é usada para destacar partes de texto dentro de um parágrafo.

(Voltar)

9. Tag <small>

A tag <small> é usada para definir texto com tamanho menor, geralmente para notas ou disclaimers.

(Voltar)

10. Tag <i>

A tag <i> é usada para enfatizar texto através de itálico.

(Voltar)

11. Tag <u>

A tag <u> é usada para sublinhar o texto.

(Voltar)

12. Tag <strong>

A tag <strong> é usada para destacar o texto com maior importância sem implicar qualquer importância implícita.

(Voltar)

13. Tag <ol>

A tag <ol> é usada para criar uma lista ordenada, onde cada item é numerado.

(Voltar)
  1. Item 1
  2. Item 2
  3. Item 3

14. Tag <ul>

A tag <ul> é usada para criar uma lista não ordenada, onde cada item é marcado com um ponto ou outro marcador.

(Voltar)
  • Item 1
  • Item 2
  • Item 3

15. Tag <li>

A tag <li> é usada para definir um item em uma lista, seja ordenada ou não ordenada.

(Voltar)

16. Tag <a>

A tag <a> é usada para criar um hiperlink para outra página, ou uma âncora em uma mesma página.

(Voltar)

17. Tag <hr>

A tag <hr> é usada para criar uma linha horizontal.

(Voltar)

18. Tag <sub>

A tag <sub> é usada para definir texto como subscrito.

(Voltar)

19. Tag <sup>

A tag <sup> é usada para definir texto como sobrescrito.

(Voltar)

20. Tag <blockquote>

A tag <blockquote> é usada para citar um bloco de texto de outra fonte.

(Voltar)
